# Product Management System
## Description

Nodejs Express TypeScript repository.

## Node version

```bash
$ v18.15.0 LTS
```

## Installation dependencies

```bash
$ npm install
```

## For knex migration install knex globally

```bash
$ npm install knex -g
$ knex migrate:latest
```


## Build and Running the app

```bash
$ npm run build
# development
$ npm run start

# watch mode
$ npm run dev
